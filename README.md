# Video Frame Capture 🎬

一个功能强大的Chrome浏览器插件，可以让你从任何网站的视频中截取并下载视频帧图片。

## ✨ 功能特点

### 核心功能
- **一键下载视频首帧** - 无需播放视频，直接下载视频开头的第一帧
- **下载指定时间帧** - 输入时间戳（如 10、1:30、02:15），自动跳转并下载该时间点的帧
- **下载当前播放帧** - 截取视频当前播放位置的帧
- **多格式支持** - 支持 PNG、JPEG、WebP 三种图片格式
- **多视频支持** - 自动检测页面中的所有视频，可选择要截取的视频

### TikTok/抖音专属功能
- **智能文件命名** - 自动提取视频ID和用户名，文件名格式：`用户名_视频ID_时间戳.格式`
- **悬浮操作面板** - 在TikTok/抖音页面显示悬浮按钮，无需打开插件弹窗
- **URL变化检测** - 自动检测滑动切换视频，实时更新视频信息

## 📥 安装方法

### 方法一：开发者模式加载（推荐）

1. **下载插件**
   - 点击上方绿色 "Code" 按钮，选择 "Download ZIP"
   - 解压下载的文件

2. **打开Chrome扩展管理页面**
   - 在Chrome浏览器地址栏输入: `chrome://extensions/`
   - 或者点击菜单 (⋮) → 更多工具 → 扩展程序

3. **启用开发者模式**
   - 在页面右上角找到"开发者模式"开关并打开

4. **加载插件**
   - 点击左上角"加载已解压的扩展程序"按钮
   - 选择解压后的 `video-frame-capture` 文件夹
   - 插件图标会出现在浏览器工具栏

## 🎯 使用方法

### 基础使用

1. **打开包含视频的网页**
   - 支持YouTube、B站、腾讯视频、TikTok、抖音等大多数视频网站
   - 也支持任何包含 `<video>` 标签的网页

2. **点击插件图标**
   - 插件会自动检测页面中的视频
   - 显示视频数量、分辨率、时长等信息

3. **选择操作**
   - **下载首帧**: 点击"一键下载视频首帧"按钮
   - **下载指定帧**: 输入时间后点击"跳转并下载指定帧"
   - **下载当前帧**: 播放视频到目标位置，点击"下载当前播放帧"

### TikTok/抖音悬浮面板

在TikTok或抖音页面，右下角会显示悬浮按钮：

```
        ┌──────────┐
        │ 🔄 刷新  │  ← 粉色按钮
        ├──────────┤
        │ 📷 首帧  │  ← 紫色按钮
        ├──────────┤
        │ 🖼️ 当前帧│  ← 绿色按钮
        └──────────┘
        
     🎬  ← 点击展开/收起面板
```

- **🔄 刷新**: 重新检测当前页面的视频
- **📷 首帧**: 下载视频首帧
- **🖼️ 当前帧**: 下载当前播放位置的帧

## ⏱️ 时间格式说明

支持以下三种时间格式：
- **秒数**: 如 `10` 表示第10秒
- **分:秒**: 如 `1:30` 表示1分30秒
- **时:分:秒**: 如 `1:02:15` 表示1小时2分15秒

## 📁 文件命名规则

### 普通网站
```
视频标题_时_分_秒.格式
```
示例：`我的视频_00_01_30.png`

### TikTok/抖音
```
用户名_视频ID_时_分_秒.格式
```
示例：`cradlefuzz0_7591440894771924254_00_05_23.png`

## ⚠️ 注意事项

1. **DRM保护视频**: 部分受DRM保护的视频（如Netflix付费内容）可能无法截取
2. **跨域限制**: 某些跨域视频可能因安全策略无法截取
3. **视频加载**: 首次使用需等待视频元数据加载完成
4. **高清视频**: 建议将视频画质调至最高以获得最佳截图质量

## 🔧 技术实现

- 使用 Canvas API 从视频元素截取帧
- 使用 Chrome Downloads API 保存图片
- 使用 MutationObserver 动态检测新视频
- TikTok/抖音智能信息提取（用户名、视频ID）
- 支持 Manifest V3 规范
